# AWS Academy Cloud Foundations

Repositório destinado ao armazenamento dos meus materiais de estudo do curso **AWS Academy Cloud Foundations**.

## 📚 Sobre

Este repositório reúne anotações, resumos, laboratórios, exercícios e demais materiais produzidos durante o estudo do curso. O objetivo é organizar o conteúdo aprendido e servir como material de consulta para futuras revisões.

## 🎯 Objetivos

- Consolidar os conhecimentos em Computação em Nuvem.
- Documentar os principais conceitos apresentados no curso.
- Armazenar exercícios e atividades práticas.
- Facilitar revisões e consultas futuras.
- Acompanhar minha evolução no aprendizado da plataforma AWS.
